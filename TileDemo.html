Groover-WebGL2D
Fullscreen tiles. webGL1 shader.

Displaying tile maps via canvas 2D can be very slow, eating away at you games frame rate. This shader renders a tile map in one pass and is very quick. It will no longer be a question of how many tiles can I render but how many tle map layers is too many?

Shader features include zoomable, repeatable, fadeable, parallaxable full screen tile maps, from both regular and irregular sprite sheets.
