Grid shader.

Grid shader is a one pass high performance grid that can be used for a variety of 2D applications. Zoom and pan

Currently the grid axis numbers and ticks are render via 2D canvas context, but I do plan on creating a separate shader to do this job in the future.

The grid can be customised, checker colour and alpha, grid line width, colour, and alpha. Lines can be displayed at 2 or 3 levels. Grid interval from 2 to 16.

Currently the grid is zooming by scaling i the vertex shader, this quickly reaches the floating point limits of the shader. Upcoming versions will be an infinite Zoom (well to the 64bit floating point limit of javascript).

Try the demo.

Batched Sprite Shader.

The aim of this shader is to render as many sprites as possible from within JavaScript.

Each sprite can be scaled, rotated, positioned and alpha blended. Sprite image source can be from fixed tile or irregular sprite sheets.

For arcade style games performance is very important. The 2D canvas API is an excellent graphics interface, easy to use flexible and fast, or should I use the words "reasonably fast". For 2D canvas rendering the same sprites I can on my machine get around 1000 on Chrome and 2000 on Firefox befor the frame rate drops below 60fps.

This shader easily out performs the 2D render by a factor of 10. The limiting factor is currently my old GPU reaching its pixel rendering limit. When rendering small sprites only I get 32,000 sprites at 60fps, when I render the full set the best I get is 16,000 at 60fps.

Try the demo. and see how many sprites your device can manage. The best I have managed is 32,000 sprites at 60FPS (Note this is a bare bones test)

Tile Map Shader.

Realtime tile map render in one pass. Renders fullscreen tile map. Map sizes are only limited by the max texture size of the GPU (which can vary between GPUs). Max tile count per pass is as many as you can fit on a single image. Will render a full screen 8 by 8 pixels tile map as quickly as a full screen 128 by 128 tile map.

With the speed of this tile map shader you can have many maps rendered at once. Each map can be scaled and translated to create awesome parallaxed tile maps.

Demo shows a 8 by 8 tiles on a map 1024 by 1024. Click and drag to paint onto the tile map.

Controls. Mouse wheel to zoom in and out, right click drag to pan.

Try the demo.
